* [[https://learnopengl.com/][learnopengl.com]] in common lisp

** run

install
- sbcl
- quicklisp
- sdl2
- sdl2-image
  
then, launch

#+begin_src
  $ git clone https://github.com/DreamAndDead/learn-opengl-in-cl.git
  $ cd learn-opengl-in-cl
  $ sbcl --load getting-started/hello-window.lisp --eval "(main)"
#+end_src

** tested with

- Linux Mint
- sbcl 2.5.11
- [ ] sdl2
- [ ] sdl2-image
